Steps to Check Logs:
Open the Grafana link for the required environment.
Login using the provided credentials.
From the left menu, go to Dashboards.
You will see the following dashboards created:
Backend Services
Frontend Services
Miscellaneous Services
Open the relevant dashboard and select the required application/pod to view logs.
Please start using Grafana for checking application logs. If you face any issues accessing the dashboards, feel free to reach out.
